# Software

Начальный набор включает, но не ограничивается:  
- **Игровой движок:** [Unity](https://unity.com/).  
- **Cреда разработки:** [Visual Studio](https://visualstudio.microsoft.com/).  
- **Графический редактор:** [paint.net](https://getpaint.net/).  
- **3D-редактор:** [Blender](https://www.blender.org/).  
- **Аудиоредактор:** [Audacity](https://www.audacityteam.org/).  
- **Система контроля версий:** [Git](https://git-scm.com/book/ru/v2/%D0%92%D0%B2%D0%B5%D0%B4%D0%B5%D0%BD%D0%B8%D0%B5-%D0%A7%D1%82%D0%BE-%D1%82%D0%B0%D0%BA%D0%BE%D0%B5-Git%3F) и облачный сервис [GitHub](https://docs.github.com/ru/get-started/git-basics/set-up-git).

Можете заменять программы на аналоги при условии легального использования.
