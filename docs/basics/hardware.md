# Hardware

Железо для стабильной работы:  
- **Операционная система:** Windows 10 (21H1 или новей).  
- **Процессор:** 64-битный CPU с поддержкой SSE2.  
- **Видеокарта:** DX10, DX11, или DX12 совместимая GPU.  
- **Оперативная память:**  16-32 ГБ RAM.  
- **Накопитель:** от 1 TB (>30 ГБ одна версия Unity + Программы + Проект 2-400 ГБ).
