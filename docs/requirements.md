# Requirements

## Hardware

Системные требования для стабильной работы:

- **Операционная система:** Windows 10 (21H1 или новей)
- **Процессор:** 64-разрядный CPU с поддержкой SSE2
- **Видеокарта:** DX10, DX11, или DX12 совместимая GPU
- **Оперативная память:**  16-32 ГБ RAM
- **Место на диске:** от 1 TB (>30 ГБ одна версия Unity + Программы + Проект 2-400 ГБ)

## Software

Начальный набор включает, но не ограничивается:

- **Игровой движок:** [Unity]
- **Cреда разработки:** [Visual Studio]
- **Графический редактор:** [Paint.NET]
- **3D-редактор:** [Blender]
- **Аудиоредактор:** [Audacity]
- **Система контроля версий:** [Git] и облачный сервис [GitHub]
